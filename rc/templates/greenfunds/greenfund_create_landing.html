Add a Green Fund

Return to Green Funds Homepage

Funds in this database are organized into four different types based on the primary source of funding. To add a new fund, please select from one of the four fund types below.

  1. Add Student Fee Fund (select if major funding comes from fees paid for by students)
  2. Add Donation Driven Fund (examples include, alumni, corporate and foundation donations)
  3. Add Department Driven Fund (this includes academic and administrative budgeted green funds)
  4. Add Hybrid Fund (select if the fund is a combination of fund types)

If you would like to add a green revolving fund, please add it to our other database here.
